Верстка сайтов

Адаптивная верстка сайтов и рассылок с использованием HTML и CSS препроцессоров (PUG, Stylus, SASS). Сборка проекта с помощью GULP. Оптимизация изображений, стилей, скриптов. Интеграция JS скриптов – настройка, использование API, написание своих несложных скриптов на JQUERY, базовые знания нативного JS.

Интеграция CMS

Интеграция верстки в MODX Revolution с использованием стандартного шаблонизатора и Femom. Работа с модулями магазина - Shopkeeper (делал коммерческие проекты) и MiniShop. Опыта разработки модулей нет. Так же изучаю интеграцию шаблонов в Wordpress и OpenCart.

Разработка дизайна

Доработка дизайна шаблона, разработка индивидуального дизайна по пожеланиям заказчика.
